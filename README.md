# Java: работа с массивами

В этом репозитории я тренируюсь решать задачи с массивами.  
Здесь собраны простые примеры: доступ к элементам, циклы, поиск min/max, сумма, фильтрация, среднее значение и другие.

---

## 📚 Содержание

| № | Файл | Что делает |
|---|------|-------------|
| 1 | `ArrayAccess` | Доступ к элементам массива (первый и последний) |
| 2 | `PrintArray` | Выводит все элементы массива через цикл |
| 3 | `SumArray` | Считает сумму всех элементов массива |
| 4 | `WordLength` | Выводит каждое слово и его длину |
| 5 | `FindMinMax` | Находит самое большое и самое маленькое число |
| 6 | `EvenNumbers` | Выводит только чётные числа из массива |
| 7 | `LongWords` | Выводит слова длиннее 4 символов |
| 8 | `AverageArray` | Считает среднее арифметическое элементов |

---

## Что я тренирую 
- Доступ к элементам массива по индексу
- Цикл `for` для обхода массива
- Поиск минимального и максимального значения
- Накопление суммы
- Фильтрация элементов по условию
- Длина строки в массиве `(length())`
- Вычисление среднего арифметического

## Зачем этот репозиторий  
Это моя шпаргалка по массивам. Когда забуду, как найти максимум или отфильтровать элементы, открываю и вспоминаю.

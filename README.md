# Java: работа с массивами

В этом репозитории я тренируюсь решать задачи с массивами.  
Здесь собраны простые примеры: доступ к элементам, циклы, поиск min/max, сумма, фильтрация, среднее значение и другие.

---

## 📚 Содержание

| № | Файл | Что делает |
|---|------|-------------|
| 1 | `ArrayAccess.java` | Доступ к элементам массива (первый и последний) |
| 2 | `PrintArray.java` | Выводит все элементы массива через цикл |
| 3 | `SumArray.java` | Считает сумму всех элементов массива |
| 4 | `WordLength.java` | Выводит каждое слово и его длину |
| 5 | `FindMinMax.java` | Находит самое большое и самое маленькое число |
| 6 | `EvenNumbers.java` | Выводит только чётные числа из массива |
| 7 | `LongWords.java` | Выводит слова длиннее 4 символов |
| 8 | `AverageArray.java` | Считает среднее арифметическое элементов |
